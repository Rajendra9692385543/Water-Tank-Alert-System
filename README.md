# Water-Tank-Alert-System
Smart Water Tank Alert System using NodeMCU (ESP8266) with dual float switches and buzzer alerts.

## Overview
This project is a smart water tank monitoring system built using NodeMCU (ESP8266).
It provides dual alerts for:
- Low Water Level
- Tank Overflow

## Hardware Used
- NodeMCU ESP8266
- 2 Side-mounted Float Switches
- Passive Buzzer
- 5V Adapter

## Features
- Dual float validation logic
- Overflow emergency sound
- Low level interval alert
- Designed for future mobile app integration

## Pin Configuration
LOW_FLOAT  -> D5
HIGH_FLOAT -> D6
BUZZER     -> D7
